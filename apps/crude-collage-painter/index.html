Keyboard Shortcuts

Modes

  • p - Select mode
  • g - Gallery mode
  • d - Draw mode

Image Operations

  • i - Rotate image source
  • x - Delete selected image (gallery mode)

Canvas Operations

  • c - Clear canvas
  • s - Save image
  • m - Paint grid

Copy Modes

  • 1 - Relative copy mode
  • 2 - Stamp copy mode
  • 3 - Absolute copy mode

Draw Mode Controls

  • ← / → - Adjust selection cursor size
  • ↑ / ↓ - Adjust zoom
  • ALT + Arrow Keys - Move source position
  • SHIFT + Arrow Keys - 10x faster adjustment

Info Box

  • ? or h - Toggle this help
  • ESC - Close this help
