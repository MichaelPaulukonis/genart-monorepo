D R A G L I N E

a different kind of text landscape

  • Mouse: Click and drag to move text blocks.
  • Arrow Keys: Move the currently selected block.
  • SHIFT + Up/Down: Change the selected block's z-index.
  • SHIFT + Left/Right: Cycle which block is selected.
  • SHIFT + ALT + S: Enter selection mode to crop a region of the canvas.
    • Arrow Keys: Nudge the selection; hold SHIFT to move two cells.
    • ENTER or SHIFT + S: Save the cropped selection.
    • ESC: Exit selection mode without saving.
  • SHIFT + S: Save a black-and-white copy of the entire canvas as dragline.YYYYMMDD.hhmmss.png.
  • SPACE: Cycle through fill characters.
  • DELETE / BACKSPACE: Remove the selected block (if more than one remains).
  • RIGHT Arrow (no block selected): Add a new block.
  • LEFT Arrow (no block selected): Remove the most recent block.
  • r: Reset text areas.
  • n: Fetch new blocks from Tumblr.
  • i: Toggle this info box. ESC: Deselects blocks or exits selection mode, and toggles this box when nothing is selected.

Check out the code at github.com/MichaelPaulukonis/dragline.git.
